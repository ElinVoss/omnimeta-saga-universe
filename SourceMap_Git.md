Load files using the exact raw URLs above. Do not use `/main/` or `/latest/`—always use the pinned commit SHA to ensure immutability.

### For Future Updates
When adding new files (N2–N30 prompts, additional dossiers, etc.):
1. Commit changes to repository
2. Get new commit SHA: `git rev-parse HEAD`
3. Update this SourceMap with new SHA-pinned URLs
4. Commit updated SourceMap

### For Canon Verification
Before any session that modifies foundation files:
```bash
git clone https://github.com/ElinVoss/omnimeta-saga-universe.git
cd omnimeta-saga-universe
sha256sum -c 05_Continuity_Ledger/hashes/v2.0-foundation.sha256.txt
```

If any fail → retcon detected → investigate before continuing
