
---

## USAGE

1. At the end of your Perplexity Space session, copy the template above
2. Replace `<paste actual commit hash here>` with the real commit SHA
3. Adjust "Files created" to list actual files generated
4. Adjust "CONTINUE FROM" to specify next task
5. Paste into session notes or handoff document

---

## WHY THIS MATTERS

The session-close format ensures:
- **Continuity**: Next session knows exactly what was done
- **Traceability**: Commit SHA pins the state
- **Clarity**: "CONTINUE FROM" prevents task fragmentation
