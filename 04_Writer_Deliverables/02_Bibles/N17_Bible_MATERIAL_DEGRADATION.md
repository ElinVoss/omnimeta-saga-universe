### Evacuation Manifest System

**Administrative Structure**:
- Numbered refugees: Each assigned identity marker (dehumanizing; bureaucratic necessity)
- Planned corridors: Routes between safe zones documented in manifest ledgers
- Route failures: When corridor cut by faction activity, marked as "route failure" with date/cause
- Arm-bands: Physical identification worn by refugees; numbered to match manifest entries
- Destination codes: Safe zones designated by alphanumeric codes (depersonalized geography)

**Manifest Language Examples**:
- "Route 7-A: Marked inactive. Corlexi fortification blocks corridor. 47 refugees redirected to Route 7-B."
- "Refugees 1847-1903: Issued arm-bands. Destination: Safe Zone K-4. Route: 9-C (provisional)."
- "Route 9-C: Cut by Mobel purge operation. Casualties: 12. Survivors redirected. Route 9-D established (temporary)."

**Dehumanization Mechanics**:
- Refugees lose names in official documentation (become numbers)
- Arm-bands physically mark displacement status
- Manifest reduces human movement to logistical problem
- Bureaucratic language distances officials from suffering
- Numbering system necessary for coordination but psychologically corrosive

**Backlash**:
- Refugees resist numbering system (want names recognized)
- Families separated by different destination assignments
- Route failures cause panic (trust in manifest coordination erodes)
- Resistance members question whether bureaucracy serves or dehumanizes

**Methods**:

- Moral weight: Each purge Jhace fails to prevent accumulates psychological burden

### Route Planning Mechanics (Primary Beat)

**Cycle Pattern**:
1. **Route planned**: Resistance scouts identify viable corridor between zones
2. **Corridor fails**: Faction activity (fortification, purge, territorial claim) blocks route
3. **New path cut**: Alternative route discovered; refugees redirected
4. **Repeat**: New route becomes primary; cycle continues

**Failure Causes**:
- Corlexi fortifications: Geometric constructions block physical passage
- Mobel purge zones: Routes pass through active cleansing operations (suicide to cross)
- Salee integration centers: Forced healing/assimilation operations trap refugees
- Eries dissolution zones: Release doctrine spread; psychological threat deters passage
- Infrastructure collapse: Buildings/bridges fail from cumulative damage; routes disappear

**Adaptation Requirements**:
- Continuous reconnaissance (Vael and scouts constantly searching new paths)
- Real-time manifest updates (routes change faster than documentation; ledgers lag reality)
- Refugee flexibility: Must adapt to route changes immediately; no time for planning
- Resource strain: Multiple routes require distributed supplies; efficiency decreases

**No Permanent Solutions**:
- Every discovered route eventually cut (cycle never breaks)
- Multi-corridor strategy spreads risk but increases complexity
- Attrition from displacement: Not all refugees survive route changes (casualties from exposure, exhaustion, faction encounters)
